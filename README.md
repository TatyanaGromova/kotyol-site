# kotyol-site

Лендинг сервисного центра "КотёлЪ" (Саткинский район): монтаж отопления, газификация и сервис котлов.

## Технологии

- HTML5
- CSS3
- Vanilla JavaScript

## Структура проекта

- `index.html` — основной HTML страницы
- `style.css` — стили и адаптив
- `script.js` — интерактив и поведение UI
- `images/` — изображения и медиа

## Быстрый старт

Проект статический, сборка не нужна.

1. Склонируйте репозиторий.
2. Откройте `index.html` в браузере.

Для локальной разработки рекомендуется запуск через простой HTTP-сервер:

```bash
python -m http.server 8080
```

После запуска сайт будет доступен на `http://localhost:8080`.

## Рекомендации по работе

- Вносите изменения отдельными небольшими коммитами.
- Проверяйте мобильную адаптацию после изменений.
- Сжимайте изображения перед добавлением в `images/`.

## Деплой

Подходит любой статический хостинг:

- GitHub Pages
- Netlify
- Vercel (static)
- Nginx/Apache
