Политика обработки персональных данных

1. Общие положения

Настоящая Политика определяет порядок обработки персональных данных пользователей сайта сервисного центра «КотёлЪ».

Используя форму заявки на сайте, пользователь может передать свои персональные данные для связи и обработки обращения.

2. Какие данные обрабатываются

Сервисный центр «КотёлЪ» может обрабатывать следующие данные:

  • имя;
  • номер телефона;
  • содержание заявки;
  • информация, указанная пользователем в поле «Комментарий».

3. Цели обработки данных

Персональные данные обрабатываются для:

  • приёма и обработки заявки;
  • связи с пользователем;
  • консультации по услугам;
  • согласования выезда мастера;
  • выполнения работ по ремонту, обслуживанию, запуску, монтажу отопления, подбору или покупке котла;
  • ведения внутреннего учёта заявок.

4. Правовые основания обработки

Обработка персональных данных осуществляется на основании согласия пользователя на обработку персональных данных.

5. Способы обработки

Обработка персональных данных может осуществляться с использованием средств автоматизации и без использования таких средств.

6. Передача данных

Персональные данные могут быть переданы мастеру или администратору сервисного центра только для обработки заявки и связи с пользователем.

Персональные данные не передаются третьим лицам для рекламы и не продаются.

7. Срок хранения

Персональные данные хранятся до достижения целей обработки или до отзыва согласия пользователем, если иное не требуется законодательством РФ.

8. Отзыв согласия

Пользователь может отозвать согласие на обработку персональных данных, направив обращение по контактам, указанным на сайте.

9. Защита данных

Сервисный центр принимает необходимые меры для защиты персональных данных от неправомерного доступа, изменения, распространения или уничтожения.
